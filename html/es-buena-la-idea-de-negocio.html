¿Es buena la idea de negocio?

Texto

¿Es buena mi idea de negocio?
Después de generar una idea de negocio, es importante comprobar si realmente tiene potencial. Para ello, podemos utilizar una pequeña lista de verificación que ayuda al alumnado a analizar sus ideas de manera sencilla y práctica.

 
1. ¿Resuelve un problema real?
¿Qué necesidad o problema estamos atendiendo?
¿A quién afecta ese problema?
¿Por qué merece la pena solucionarlo?
 
2. ¿Quién sería el cliente o usuario?
¿Quién utilizaría esta idea? (alumnado, familias, profesorado, barrio…)
¿Por qué esa persona estaría interesada?
 
3. ¿Qué aporta de nuevo?
¿Es diferente a lo que ya existe?
¿Mejora algo actual?
¿Aporta comodidad, rapidez, ahorro, algo más sostenible…?
 
4. ¿Es viable?
¿Tengo (o puedo conseguir) los recursos mínimos para empezar?
¿Requiere algo demasiado costoso o difícil de obtener?
¿Podríamos desarrollarla como prototipo en el centro?
 
5. ¿Nos motiva trabajar en ella?
¿Nos gusta la idea?
¿Estaríamos dispuestos a continuar desarrollándola?
¿Es una idea que sentimos “nuestra”?
 
Si la mayoría de las respuestas son positivas, probablemente estamos ante una buena base para un futuro proyecto emprendedor. Este análisis es perfecto para completar después de usar herramientas como Brainstorming, Seis sombreros para pensar o Phillips 66.
